# Marketing Assets

Acest director contine imagini si resurse pentru landing page-ul public:

- `dashboard-preview.png` - Screenshot dashboard pentru hero section
- `feature-*.png` - Icoane/imagini pentru sectiunea caracteristici
- `testimonials/` - Poze testimoniale clienti
- `logos/` - Logo-uri parteneri

## Note
- Imagini optimizate pentru web (max 500KB)
- Format recomandat: PNG/WebP pentru transparenta, JPG pentru poze
- Dimensiuni recomandate dashboard preview: 1200x800px
