# Instrucțiuni pentru screenshot dashboard

Pentru a adăuga screenshot-ul dashboard-ului pe landing page, urmează acești pași:

## Metoda 1: Screenshot manual (recomandat)

1. Deschide dashboard-ul în browser (`http://localhost/fleet-management/dashboard`)
2. Fă un screenshot (Windows: Win + Shift + S, sau PrtScn)
3. Salvează imaginea ca: `c:\wamp64\www\fleet-management\assets\images\marketing\dashboard-preview.png`
4. Dimensiuni recomandate: 1200x800px
5. Optimizează imaginea (compresie PNG sau convertește la WebP pentru încărcare mai rapidă)

## Metoda 2: Folosind tool de screenshot

PowerShell (necesită Snipping Tool sau browser extension):
```powershell
# Sau folosește un tool de screenshot automat
# Start-Process "ms-screenclip:"
```

## Metoda 3: Voi crea un SVG placeholder stilizat

Dacă vrei temporar, pot crea un SVG placeholder care arată ca un dashboard real cu carduri și butoane.

---

**Care metodă preferi?**
1. Faci tu screenshot-ul manual și îl salvezi (cea mai bună calitate)
2. Creez un SVG placeholder profesional temporar până faci screenshot-ul real
